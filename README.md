# Twitter Clone (Revamped)

A static front-end recreation of Twitter's core timeline UI — post a tweet to a live-updating timeline, toggle dark mode, and a separate login page — built as a JavaScript DOM manipulation practice project.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat-square&logo=html5&logoColor=white) ![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat-square&logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat-square&logo=javascript&logoColor=black)

> 📚 Learning project — not affiliated with X Corp / Twitter. Not a full clone: focuses on posting tweets to the DOM and a dark-mode toggle.

## Features

- Post a tweet to a live timeline (prepended, newest first)
- Basic empty-tweet validation
- Dark mode toggle
- Separate login page (`login.html`) with its own styles/script

## Tech Stack

HTML5 · CSS3 · Vanilla JavaScript (DOM manipulation, no framework)

## Getting Started

```bash
git clone https://github.com/phiwakonkem/Twitter.git
cd "Twitter/Revamped Twitter Clone"
```

Open `index.html` directly in your browser, or use the VS Code **Live Server** extension for auto-reload.
